# Isolated-USB-To-UART
### A Simple 5kV Isolated USB To UART Bridge

The isolated USB to UART Bridge works with Baudrates from 300 baud to 3 Mbaud.</br>
It can withstand 5000Vrms for 1 minute.</br>
And 565Vpeak Continuously, without destroying your computer.</br>

It can opperate with a circuit voltage from 1.8V to 5V.
